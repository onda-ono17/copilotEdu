  1. 1단계 신청 서비스 선택
  2. 2단계 신청 정보 입력
  3. 3단계 신청 내용 확인
  4. 4단계 제출 완료

처리 단계 및 확인사항

신청이 제출되면 접수 완료, 검토 중, 보완 요청, 최종 승인 순서로 상태가 갱신되며 주문 목록에서 같은 번호로 추적할 수 있습니다.

  • 보완 요청이 발생하면 신청 시 입력한 연락처와 전자우편으로 안내하는 구조를 권장합니다.
  • 표의 금액과 수량처럼 비교가 필요한 숫자 데이터는 오른쪽 정렬해 가독성을 높였습니다.
  • 모바일에서는 표 영역을 가로로 이동할 수 있게 두고, 입력폼은 한 열로 재배치되도록 구성했습니다.
